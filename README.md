# ChatBot_Facebook_Page
Chatbot for Facebook page.

ใช้ในการโต้ตอบกับข้อความของแอดมิน หรือระบบบอทอื่นๆ เพื่อเพิ่มFunction การใช้งานที่บอทสำเส็จรูปไม่มี
สามารถนำไปพัฒนาต่อยอดได้
